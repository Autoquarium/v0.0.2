Autoquarium Documentation
    The Autoquarium is a standalone module that can be mounted on the edge of a fish tank. 
    The module will be designed such that it can be installed on many aquarium sizes and can 
    be transferred between different fish tanks easily. 

    Including a pH and temperature sensor, the Autoquarium measures and monitors the ecosystem 
    of an aquarium. With the provided dashboard, users can check in on their aquatic pets from 
    anywhere in the world.

    This documentation is the code running on the ESP32 processor inside the Autoquarium. As such, 
    this code overviews how our device establishes a connect to our cloud server, reads and publishes 
    sensor values over MQTT, sends notifications to users when dangerous water values are detected, 
    and gets user commands from the dashboard such as changing led light brightness or triggering 
    the fish to be fed.
